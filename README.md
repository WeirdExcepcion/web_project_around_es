🌎 Around The U.S. — Web Project

Proyecto desarrollado durante el programa de desarrollo web

🧠 Descripción

Around The U.S. es una aplicación web interactiva tipo red social donde los usuarios pueden compartir lugares, interactuar con contenido y gestionar su perfil.

A lo largo de su desarrollo, el proyecto evolucionó desde manipulación básica del DOM hasta una aplicación conectada a un backend con autenticación de usuarios y persistencia de datos.

🔐 Autenticación

La aplicación cuenta con un sistema de acceso seguro:

Registro de nuevos usuarios
Inicio de sesión
Protección de rutas privadas
Persistencia de sesión mediante JWT
Cierre de sesión

Solo los usuarios autenticados pueden acceder al contenido principal.
